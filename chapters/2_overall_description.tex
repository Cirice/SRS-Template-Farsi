\chapter{توصیف کلی}
\clearpage

\noindent { 
در این فصل به جزییات کلی معماری  سیستم تحت توسعه پرداخته خواهد شد.
 ضمنا انواع کاربران و ذی‌نفعان نرم‌افزار نیز معرفی خواهند شد.
 جزییات تعامل بقیه سامانه‌‌ها با سیستم نیز شرح داده خواهد شد.
همچنین فرض‌ها و محدودیت‌های پیش روی توسعه نرم‌افزار معرفی می‌شوند.
}

\section{توصیف محصول}
\noindent { 
این قسمت حاوی توصیف نرم‌افزار به عنوان یک محصول است.
}

\section {کاربری‌های محصول}
\noindent {
این بخش حاوی انواع کاربری‌هاییست که برای نرم‌افزار در نظر گرفته شده است.
}

\section {انواع کاربران}
\noindent {
در این قسمت انواع کاربران درگیر با سیستم و چگونگی تعاملشان با سیستم
معرفی می‌شوند.
}

\section {محدودیت‌ها}
\noindent {
در این قسمت حاوی محدودیت‌هایست که برای توسعه‌ی سیستم در راه رسیدن به
کاربری‌ها در نظر گرفته شده وجود دارد.
}

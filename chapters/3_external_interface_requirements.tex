\chapter{نیازمندی‌های اتصالات خارجی}
\clearpage
\noindent {
این قسمت حاوی جزییات تعامل سیستم تحت توسعه با دیگران(دیگر سیستم‌ها، کاربران و ..) است.
برای مثال در صورتی که نرم‌افزار تحت توسعه قسمتی از سیستم بزرگتریست باید جزییات کامل تعامل آن با دیگر سیستم‌ها
ذکر شود.
}

\section {رابط کاربری}
\noindent {
ساختار و جزییات رابط کاربری نرم‌افزار باید در این بخش با دقت زیاد توضیح داده شود.
}
\clearpage
\section {رابط سخت‌افزاری}
\noindent {
در صورت وجود وابستگی‌ها به سخت افزار خاصی برای اجرای درست فعالیت سیستم، در این قسمت باید
جزییات کامل این وابستگی‌ها ذکر شوند.
}

\section {رابط نرم‌افزاری}
\noindent {
در صورت وجود وابستگی به سیستم‌های نرم‌افزاری دیگر برای انجام کار‌های محول شده به سیستم تحت توسعه،
در این قسمت باید این وابستگی‌هایی را توضیح داده شوند.
}
